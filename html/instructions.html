Инструкции по настройке бота

1. Установите Node.js

Скачайте и установите Node.js с официального сайта: https://nodejs.org/

2. Скачайте код бота

Скачайте архив с кодом бота или клонируйте репозиторий:

git clone https://github.com/your-username/max-bridge.git

3. Установите зависимости

Перейдите в папку с ботом и установите необходимые пакеты:

npm install

4. Настройте бота

Создайте файл .env и добавьте в него настройки:

MATRIX_HOMESERVER=https://matrix.example.com
MATRIX_USER_ID=@max_bot:example.com
MATRIX_ACCESS_TOKEN=your_matrix_access_token
MATRIX_ROOM_ID=!room_id:example.com

TELEGRAM_BOT_TOKEN=your_telegram_bot_token
TELEGRAM_CHAT_ID=your_telegram_chat_id

5. Запустите бота

Запустите бота с помощью команды:

npm start

Или для разработки:

npm run dev

6. Проверьте работу

Отправьте сообщение в Max и проверьте, что оно появляется в Matrix и Telegram.
